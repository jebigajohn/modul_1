Position

Was ist Position?

Mit den CSS-Eigenschaften position wird gesteuert, wo die Elemente auf einer Seite dargestellt werden.

Wie funktioniert das?

postion legt fest, wie die Elemente in unserem Dokument positioniert werden. Die Eigenschaften sind top bottom left und right

position: static

Standartwert für jedes HTML Element, das Element befindet sich an einer Standartposition

static

position: relative

Das Element bleibt an seiner ursprünglichen Stelle, kann von dort aus aber mit top, left usw. verschoben werden

relative

position: absolute

Das Element orientiert sich am nächstgelegenen nicht-statischen Eltern Element, wenn es ein solches Element nicht gib, orientiert es sich am HTML-Element body

Best practice: Das Eltern-Element sollte die Eigenschaft => position: relative erhalten

absolut

position: fixed

Das Element orientiert sich am Browserfenster, es wird aus dem Dokumenten Fluss entfernt und bleibt beim scrollen immer an seiner Position

fixed

position: sticky

Das Element scrollt mit, bis eine seiner Kanten die durch topbottom leftright angegebenen Koordinate erreicht hat

sticky
