ul vs. ol

  • unorderedList
    • Das HTML Element für unorderedList nennt sich: "ul"
      • Innerhalb des HTML Element "ul" verwendet man das HTML Element "li", welches verwendet wird um innerhalb "ul" eine ungeordnete auflistung darzustellen.
    • Der Präfix dieser unorderedList sind "Bullet points"
    • Beispeile:
      • Terminal
      • Live Mode
      • UTF-8
    • Weiterführende Infos
  1. orderedList
    1. Das HTML Element für orderedList nennt sich: "ol"
      1. Innerhalb des HTML Element "ol" verwendet man das HTML Element "li", welches verwendet wird um innerhalb "ol" eine geordnete nummerierte auflistung darzustellen.
    2. Der Präfix dieser orderedList sind "Zahlen"
    3. Beispeile:
      1. Seite
      2. Seite
      3. Seite
